## Installation

Sniffies is distributed as a userscript. To run it you need a userscript manager:

- **iOS (Safari):** Install the [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887) app, then enable the Safari extension in Settings → Safari → Extensions.
- **Desktop (Chrome/Firefox):** Install the Chrome extension build
  - If you want to install this userscript instead of the Chrome extension build, install [Tampermonkey](https://www.tampermonkey.net/).

Once your userscript manager is set up, tap the button below and it will prompt you to install.
