## Installation

There are two builds: a userscript for iOS and a Chrome extension for desktop.

### iOS (Safari)

1. Install the [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887) app
2. Enable it in **Settings → Safari → Extensions**
3. Click on the install button below to go to the userscript page. On that page, click on the button next to the URL (highlighted in red in the screenshot below), then click on the userscript extension. Userscript will recognize the file and prompt you to install it. Click **Install**.

### Chrome (Desktop)

The extension isn't on the Chrome Web Store, so installation is manual:

1. Download the zip below and unzip it
2. Go to `chrome://extensions` and enable **Developer mode** (top right)
3. Click **Load unpacked** and select the unzipped folder
